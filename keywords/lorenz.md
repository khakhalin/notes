# Lorenz attractor

#chaos

Parents: [[complexity]] / [[chaos]]

A classic dynamic system:

$\displaystyle \begin{cases} dx/dt = p(y-x) \\  dy/dt = -xz + rx -y \\ dz/dt = xy-bz \end{cases}$

with (standard values, used by Lorenz himself): p = 10, r=28, b = 8/3.
