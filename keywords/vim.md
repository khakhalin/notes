#tools


The famous weird editor that everyone hates or loves.

Main commands:
`:q!` followed by `Enter` - exit without saving
`ZZ` - save and exit
`a` - start inserting (kinda like "normal editing" mode)
`esc` - end the current mode (including inserting mode)
